# BÀI TẬP 4 — SENSE HAT EMULATOR

## Nền tảng
- Raspberry Pi OS (GUI)
- Sense HAT Emulator
- Python

---

## 1. Hiển thị text
- Hiển thị "Hello IoT!" chạy ngang LED matrix
- Màu xanh lá

---

## 2. Vẽ biểu tượng
- Tự thiết kế biểu tượng: ❤️ (trái tim)
- Hiển thị trong 5 giây

---

## 3. Đọc cảm biến
- Nhiệt độ (Temperature)
- Độ ẩm (Humidity)
- Áp suất (Pressure)
- Cập nhật mỗi 1 giây

---

## 4. Dashboard LED
- Hàng 0–2: nhiệt độ (đỏ)
- Hàng 3–5: độ ẩm (xanh)
- Hàng 6–7: trạng thái
  - Xanh: bình thường
  - Vàng: cảnh báo
  - Đỏ: nguy hiểm

---

## 5. Mini game joystick
- Điều khiển pixel bằng joystick
- Ăn mục tiêu để tăng điểm
- Có hiệu ứng flash khi ghi điểm

---

## Demo
- Kéo slider trên emulator để thay đổi temp/humidity
- Điều khiển joystick để chơi game
